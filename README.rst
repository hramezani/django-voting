=============
Django Voting
=============

This is a generic voting application for Django projects


Installation
============

1. Install the ``django-voting`` distribution

2. Add ``voting`` to your ``INSTALLED_APPS``::

    INSTALLED_APPS = [
        …
        'voting',
        …
    ]

3. Run ``django-admin migrate``
